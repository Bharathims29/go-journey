# Modules 1 to 7: Practice Tasks

1. Create a Go program that prints your name, city, learning goal, and today's date in `YYYY-MM-DD` format.

2. Create a student score calculator using variables, constants, arithmetic operators, and `if/else` to print total marks, average marks, grade, and pass/fail status.

3. Create a number analyzer that uses functions to check whether a number is positive, negative, zero, even, odd, and prime.

4. Create a contact book using a map where the key is the contact name and the value is the phone number; add contacts, update one contact, search one contact, delete one contact, and print all contacts.

5. Create a `BankAccount` struct with fields for owner name and balance; add methods for deposit, withdraw, and display balance, then use those methods in `main`.
